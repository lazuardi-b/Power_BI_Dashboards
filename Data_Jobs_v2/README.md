# Data Jobs Dashboard Analysis (Version 2 - Refined)

## Project Overview

This dashboard provides a focused analysis of the data job market, highlighting key insights related to job demand, salary trends, and high-value skills. Built using Power BI, it is designed to present the most relevant information in a clear and efficient way to support better decision making.

Unlike the first version, which emphasizes broad exploration, this version prioritizes clarity and insight. The dashboard is structured as a single page report, reducing complexity while focusing on the most important metrics and relationships within the data.

The goal of this version is to move beyond exploration and deliver a more refined, user-centered analytical experience that communicates insights more effectively.

## Key Insights 💡

| Category              | Insight                                                                 |
|----------------------|-------------------------------------------------------------------------|
| **Market Size**      | The dataset includes over 479K job postings, indicating a large and active data job market. |
| **Skill Complexity** | Job postings require an average of 4.8 skills, reflecting increasing expectations for multi-skilled professionals. |
| **Core Skills**      | Python and SQL dominate the market, appearing in nearly half of all job postings. |
| **Cloud Demand**     | AWS and Azure are among the most in-demand technologies, highlighting the importance of cloud expertise. |
| **Salary Benchmark** | The median salary is approximately $113K annually ($48/hour), providing a baseline for compensation. |
| **Top Paying Roles** | Senior Data Scientist and Machine Learning Engineer roles offer the highest salaries, exceeding $150K annually. |
| **Skill vs Salary**  | Higher salaries are associated with specialized and engineering-focused roles rather than generalist positions. |

These insights highlight a competitive and evolving data job market, where strong foundational skills like Python and SQL are essential, while higher salaries are driven by specialization in engineering and cloud-related roles. 

The data also suggests that employers increasingly expect multi-skilled candidates, making both breadth (multiple tools) and depth (specialized expertise) important for career growth.

## Dashboard Overview

![Dashboard Overview](/images/Project2_Page1.png)

The dashboard is designed as a single-page report to present key insights in a clear and efficient layout. It focuses on the most relevant aspects of the data job market, including job volume, skill demand, and salary distribution.

Key metrics are displayed at the top to provide a quick summary, followed by visualizations that highlight in-demand skills and top-paying roles. Interactive filters allow users to explore the data by job title and location, enabling more targeted analysis.

## Key Features

- **Insight-Driven Structure**  
  The dashboard is organized to support key findings first, allowing users to quickly understand the most important trends before exploring the data in detail.

- **Clear Visual Hierarchy**  
  Key metrics are placed at the top, followed by supporting charts, creating a logical flow that improves readability and decision-making.

- **Focused Data Selection**  
  Only the most relevant metrics and visuals are included to avoid clutter and ensure clarity.

- **Interactive Exploration**  
  Filters for job title and location allow users to customize the view and explore specific segments of the job market.

- **Improved Readability**  
  Consistent formatting and simplified visuals make the dashboard easier to interpret compared to the previous version.

## Approach

This dashboard was developed with a focus on transforming raw job posting data into clear and actionable insights:

- **Data Preparation:** Cleaned and transformed the dataset using Power Query to ensure consistency and usability.
- **Data Modeling:** Applied a structured data model to support accurate aggregation and efficient filtering across visuals.
- **Analytical Focus:** Selected key metrics and dimensions that directly support insight generation, particularly around skills, salary, and market demand.
- **Dashboard Design:** Built a single-page layout to prioritize clarity, reduce cognitive load, and improve the overall user experience.

## Tools Used

- **Power BI:** Dashboard development and data visualization  
- **Power Query:** Data cleaning and transformation  
- **DAX:** Measures and calculations for analysis  

## Conclusion

This project demonstrates the progression from exploratory analysis to a more focused and insight-driven approach. While the first version emphasizes broad data exploration, this version refines the analysis to highlight the most meaningful patterns within the data job market.

The key takeaway is the importance of balancing foundational skills with specialized expertise. As the market evolves, professionals who combine strong core knowledge with in-demand technical skills—*particularly in engineering and cloud technologies*—are better positioned for higher-value opportunities.

Overall, this version reflects a shift toward more intentional analysis and clearer communication of insights.
